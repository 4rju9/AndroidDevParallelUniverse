## 🛠️ How to Traverse the Multiverse

Don't clone the whole repo unless you want to break the space-time continuum. To pull a specific dimension to your local machine:

1. Clone the lightweight Nexus (Main):
   ```bash
   git clone https://github.com/4rju9/AndroidDevParallelUniverse.git
   ```
2. Teleport to your desired branch:
   ```bash
   git checkout WheelPicker
   ```
3. Or, clone only the branch you need from the start:
   ```bash
   git clone --single-branch --branch wheelPicker https://github.com/4rju9/AndroidDevParallelUniverse.git
   ```
